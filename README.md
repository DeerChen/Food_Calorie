# Food_Calorie

## Introduction

使用[Scrapy](https://scrapy.org/)爬取[Hi 运动食品库](https://food.hiyd.com/)的食物卡路里热量表。

## Installation

```bash
pip install scrapy requests beautifulsoup4
```

## Usage

```bash
scrapy crawlall # 运行全部
scrapy crawl staple_food # 运行单例
```
